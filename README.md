# 15 PRIMAVERAS - Página Web para Fiestas de Quince Años

Proyecto web desarrollado para la empresa "15 PRIMAVERAS", dedicada a la organización integral de fiestas de quince años.
Incluye secciones de paquetes de banquetes, salones, vestidos, además de paneles de acceso para clientes y administradores.
Todo el diseño se construyó exclusivamente con **Bootstrap 5.3.8**, sin estilos personalizados adicionales.

### Clases principales de Bootstrap 5 utilizadas:
- Sistema de cuadrícula: `container`, `row`, `col-md-*`, `col-lg-*`
- Navegación: `navbar`, `navbar-expand-lg`, `dropdown`
- Componentes: `card`, `btn`, `form-floating`, `modal`, `alert`, `badge`
- Utilidades: `d-flex`, `align-items-center`, `shadow-sm`, `rounded-3`, `text-success`
- Elementos interactivos: `nav-tabs`, `table-responsive`, `form-check`

El proyecto es totalmente responsivo y funciona correctamente en dispositivos móviles, tabletas y computadoras.
